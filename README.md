Gulp boilerplate
======================
Fully featured web app boilerplate 

## Features
- Concatenation, Uglify
- HTML minification
- SASS, CSSO, Autoprefixr
- HTTP Server (using express)
- Livereload

## Installation & Prerequisites
Stubby
`$ npm install -g stubby`

SASS
`$ gem install sass`

Install dependencies
`$ npm install && bower install`
